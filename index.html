02 How it works

1

Install Python

Download Python from python.org. During setup check "Add Python to PATH".

2

Download DesktopToggle

Click the Download button below. You'll get a .zip with everything needed.

3

Run the installer

Extract the zip, double-click INSTALLA_DOPPIO_CLICK.bat. Accept admin permissions. Done in 30 seconds.

4

Ready!

The icon appears in the system tray. From now on it starts automatically every time Windows boots.
